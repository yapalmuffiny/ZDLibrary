# ZDLibrary
### powered by me :)

First of all, this plugin is a library. Not a public tool.

Anyhow, lets show you how to install.

1 ) Download the latest release on the GitHub releases page.

2 ) Unzip it within your host or computer.
> This should be formatted /plugins/ZDLibrary/main.js
> you should see the main.js here, not ZDLibrary/ZDLibrary!!

3 ) Power on the bot via `npm start` or `node index.js`

Thats it!
